## Project Overview
Trade Hub is a modern,  web platform that allows users to explore global products, manage their exports, and easily import products into their personal â€œMy Importsâ€ section with a single click. The platform features a clean and responsive user interface, real-time synchronization with the backend, and secure user authentication. Users can browse all products, view detailed product information, import products while respecting stock limits, and manage their own exports and imports efficiently.

## Features
- **User Authentication:** Login & Registration using Email/Password and Google OAuth  
- **Protected Routes:** Private pages accessible only to authenticated users  
- **Home Page:** Banner/slider with latest 6 products displayed in a responsive grid  
- **All Products Page:** Browse all products with search functionality and â€œSee Detailsâ€ button  
- **Product Details Page:** View full product information and import via modal  
- **Import Functionality:** Import products with quantity validation; real-time stock updates  
- **My Imports Page:** List of imported products with remove and see details options  
- **My Exports Page:** Add, update, and delete exported products  
- **Add Export/Product Page:** Submit new products for export with full form  
- **Dark/Light Mode:** Toggle between dark and light themes  
- **Dynamic Page Titles:** Each route displays a dynamic, descriptive page title  
- **Responsive Design:** Mobile, tablet, and desktop optimized layout  
- **Role-Based Access (Optional):** Different access for exporter and importer  
- **Real-Time Feedback:** Toast notifications for success, errors, and warnings  
- **Client-Side Validation:** Prevent invalid imports or form submissions  
- **Smooth Animations:** Framer Motion, AOS, and Lottie animations enhance UX
